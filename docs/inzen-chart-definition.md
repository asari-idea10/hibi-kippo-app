# 陰占プレビュー定義 v0

共有スプレッドシート「陰占」の構造を、開発者画面と将来の鑑定プロンプトで使えるように整理した定義です。

## 列

| キー | 表示 | 年齢域 | テーマ |
| --- | --- | --- | --- |
| `year` | 年柱 | 初年期 (0〜20歳頃) | ルーツ、第一印象、社会に対する表面的な顔 |
| `month` | 月柱 | 青年期 (20〜40歳頃) | 社会性、仕事、オフィシャルな立場 |
| `day` | 日柱 | 中年期 (40〜60歳頃) | 自分自身の核、プライベート、配偶者 |
| `time` | 時柱 | 晩年期 (60歳以降) | 最終的な結果、子供・部下、未来 |

## 行

| キー | 表示 | 意味 |
| --- | --- | --- |
| `heavenly_stem` | 天干 | 表面に現れる性質、意識、看板 |
| `earthly_branch` | 地支 | 現実の環境、肉体、無意識の行動パターン |
| `hidden_stems_main` | 蔵干 主 | 本元。最も強く発揮されるメインの才能 |
| `hidden_stems_middle` | 蔵干 中 | 中気。環境や状況によって引き出されるサブスキル |
| `hidden_stems_extra` | 蔵干 余 | 初気。無意識のベースとなる潜在的な気質 |
| `nine_stars` | 九星 | その柱が持つエネルギーの波、タイミング |
| `story` | ストーリー | 天干・地支・蔵干を統合した、その時期/領域における人生の物語 |

## UI方針

- 開発者画面では、検索フォーム下に陰占プレビューを表示する。
- 年・月・日は共通暦エンジンの計算結果を表示する。
- 時間列は、時柱・時盤・真太陽時補正を実装するまでは未接続として表示する。
- 各セルには、定義JSONの `meaning` と `challenge` を参照したツールチップを付ける。
- ストーリー行は、初期状態では「（ここに物語が生成されます）」を表示する。

## データ

実装データは `src/data/inzen-chart-definition.json` を正とします。
