# 主要選日 こよみページ寄せ方針 v0

## 結論

主要選日の計算は、スプシマスターが不足しているため、こよみのページの出力結果を検証正本として寄せる。

ただし、こよみのページ全期間データを丸ごと持つのではなく、日々吉方側で計算できるものは計算し、重要サンプルで照合する。

## 正本の扱い

優先順位は次の通り。

1. こよみのページで確認した正本サンプル
2. 手元万年暦で確認した正本サンプル
3. スプシに残っている legacyRaw / フラグ
4. 日々吉方側の計算ルール

正本サンプルがある日は、計算結果より正本を優先する。

日々吉方では、ここを検証正本として扱い、まずは一粒万倍日・三隣亡・不成就日を候補計算として実装した。

## 計算候補 v0

### 一粒万倍日

こよみページの一粒万倍日 `方式I` に寄せ、節月と日支で判定する候補を実装。

例:

```text
寅月: 丑・午
卯月: 酉・寅
```

2021-02-03 は寅月・午日なので候補計算では一粒万倍日に該当する。

### 三隣亡

こよみページの三隣亡説明に寄せ、節月と日支で判定する候補を実装。

```text
正月・四月・七月・十月: 亥の日
二月・五月・八月・十一月: 寅の日
三月・六月・九月・十二月: 午の日
```

2021-02-03 は寅月・午日なので候補計算では三隣亡に該当しない。

### 不成就日

こよみページの不成就日説明に寄せ、旧暦月日で判定する候補を実装。

```text
旧暦1月・7月: 3, 11, 19, 27日
旧暦2月・8月: 2, 10, 18, 26日
旧暦3月・9月: 1, 9, 17, 25日
旧暦4月・10月: 4, 12, 20, 28日
旧暦5月・11月: 5, 13, 21, 29日
旧暦6月・12月: 6, 14, 22, 30日
```

2021-02-03 は旧暦12月22日なので候補計算では不成就日に該当する。

## 注意

主要選日は、ひとつの式で全部出せるものではない。

干支、節月、旧暦、期間暦注、下段暦注を分けて扱う。ここを分けることで、後から流派差や正本差分が出ても調整できる。
